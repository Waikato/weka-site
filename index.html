Hi. I'm WEKA.

Weka is an open source (GPL) machine learning software in JAVA. It contains tools for data preparation, classification, regression, clustering, association rules mining, and visualization.

Why Weka?

It is the most popular open source software for Machine Learning in Java.

It is the most popular tool to learn Machine Learning, thanks to the best-selling book "Data Mining" and MOOC courses.

It has more than 10 milion downloads.

It has been cited in more than 18,000 research and applied data science publications.

Learn easily Machine Learning with WEKA

WEKA can be used to build ML pipelines, train classifiers, and run evaluations, from the Graphical Uner Interface, without having to program.

Open a Dataset

First, we open the dataset that we would like to evaluate. WEKA uses arff files, its own specific file format designed exclusively for machine learning and data mining tasks.

Choose a classifier

Second, we need to select a classifier to use. J48 is the standard decision tree classifier, that is a 100% explanaible classifier.

Get the performance result

Finally, we run a 10-fold cross-validation evaluation, and we get the performance result of the classifier selected previously.

Deep Learning with WEKA

WekaDeeplearning4j is a new deep learning package for Weka. It is developed to incorporate the modern techniques of deep learning into Weka. The backend is provided by the Deeplearning4j Java library.

WEKA Interoperability

WEKA can be integrated with the most popular data science tools.

R

You can use Weka from R using the RWeka package, or otherwise, you can use R from Weka using the RPlugin.

Python

It is possible now for WEKA to interact with CPython (Scikit-learn).

Spark

The distributed Weka for Spark package has been available in Weka for several years now. One nice thing about the package is that it allows any Weka classifier to be trained in Spark.
